## Installation

DQLoopBanner is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
source 'https://github.com/jjjjjeffrey/DQPodSpecs.git'
platform :ios, '8.0'
use_frameworks!
pod 'DQLoopBanner', '~> 1.0.0'
```
